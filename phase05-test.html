🔍 Test Rule for Beginners

Observe more than you explain. Let the user struggle a little — that's where the real learning is.

Prepare first

  • Your finished Phase 04 prototype
  • At least 1 real target user to test with
  • A simple task to ask each tester to try
  • Facilitator name and date

How to complete

  • Complete T12 with at least 1 real tester
  • Complete T13 with the main issue you found
  • Complete T14 with all 8 pitch slides
  • Use the guide dropdown when stuck
  • Score at least 3/5 in the quiz
